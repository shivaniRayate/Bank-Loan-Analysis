# 📊 Bank Loan Analysis Dashboard

## Project Overview

This project analyzes bank loan data to understand lending performance, loan distribution, customer behavior, and default trends. The goal is to create an interactive dashboard that helps stakeholders monitor key business metrics and make data-driven decisions.

---

## Business Problem

Banks need to monitor loan performance and identify factors affecting loan approvals and defaults. This dashboard provides insights into loan portfolio performance across different branches, states, and customer segments.

---
